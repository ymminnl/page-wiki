# Helpers
Ayuda a usuarios y moderación preventiva del chat.

---

## Gestión de Sanciones

*   **`/warn <jugador> <plantilla>`**: Emite una advertencia registrada en el historial del usuario.
*   **`/kick <jugador> <razón>`**: Expulsa a un jugador del servidor (Desconexión forzada).
*   **`/mute <jugador> <plantilla>`**: Aplica silencio temporal. La duración es calculada por la plantilla.
*   **`/freeze <jugador>`**: Inmoviliza a un jugador para revisión o chat administrativo.
*   **`/warnings <jugador>`**: Muestra la lista activa de advertencias de un usuario.
*   **`/checkwarn <jugador>`**: Verifica el estado de una advertencia específica.

### Plantillas Comunes
*   `Mayusculas`
*   `Advertencia-General`
*   `Spam`
*   `Toxicidad-Leve`
*   `Publicidad`

---

## Investigación

*   **`/history <jugador>`**: Muestra el historial completo de sanciones del usuario.
*   **`/checkban <jugador>`**: Verifica si un usuario tiene un bloqueo activo.
*   **`/checkmute <jugador>`**: Verifica si un usuario está silenciado actualmente.
*   **`/dupeip <jugador>`**: Escaneo de coincidencias de IP para detectar multicuentas.
*   **`/iphistory <jugador>`**: Muestra el historial de direcciones IP usadas por el usuario.
*   **`/geoip <jugador>`**: Muestra la ubicación geográfica aproximada de la conexión.
*   **`/litebans help`**: Muestra la ayuda general del sistema de sanciones.

---

## Supervisión

*   **`/vanish`**: Activa el modo invisible para monitorear sin ser visto.
*   **`/socialspy`**: Permite leer los mensajes privados (`/msg`) de otros usuarios en tiempo real.
*   **`/staffchat`**: Canal de chat exclusivo para comunicación interna del staff.
*   **`/invsee <jugador>`**: Permite abrir y visualizar el inventario de un usuario.
*   **`/enderchest <jugador>`**: Permite abrir y visualizar el cofre de ender de un usuario.

---

## Movimiento

*   **`/tp <jugador>`**: Teletransportarse instantáneamente a la ubicación de un jugador.
